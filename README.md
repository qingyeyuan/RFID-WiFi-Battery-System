# RFID+WiFi智能电池管理系统

> ESP32 电池管理系统：RFID身份识别 + WiFi数据上传 + 环境监测

## 🚨 关键约束

```
⚠️ MFRC522 只能用 3.3V，接 5V 会烧毁
⚠️ 所有组件 GND 必须共地
⚠️ INA226 分流电阻值必须与实际硬件匹配
```

## 🚀 快速开始

### 1. 硬件连接

**核心引脚映射：**

| 组件 | 信号线 | GPIO | 总线 |
|------|--------|------|------|
| MFRC522 | SCK/MISO/MOSI/SS/RST | 18/19/23/5/27 | SPI |
| SHT3X + OLED | SDA/SCL | 32/33 | I2C1 |
| INA226 | SDA/SCL | 21/22 | I2C2 |
| 蜂鸣器 | I/O | 26 | - |
| Boot按钮 | 按键 | 0 | - |

**完整接线表见下方 [硬件接线表](#-硬件接线表)**

### 2. 首次配网

```
1. 上传代码到 ESP32
2. 连接 WiFi 热点 "Device-Config"（密码：12345678）
3. 浏览器自动跳转配置页面
4. 填写配置项（见下方配置说明）
5. 保存后自动重启
```

### 3. 配置项说明

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| WiFi名称/密码 | 连接的网络 | - |
| 服务器IP/端口 | TCP服务器地址 | 192.168.1.100:8080 |
| 设备ID | 设备唯一标识 | 25824 |
| **分流电阻值** | INA226分流电阻(Ω) | 0.1 |
| **温度上限** | 报警阈值(°C) | 30.0 |
| **温度下限** | 报警阈值(°C) | 0.0 |
| **湿度上限** | 报警阈值(%) | 80 |
| **湿度下限** | 报警阈值(%) | 20 |

**分流电阻选择参考：**

| 电流范围 | 推荐阻值 |
|----------|----------|
| 0-1A | 0.1-0.5Ω |
| 1-5A | 0.05-0.1Ω |
| 5A以上 | 0.01-0.05Ω |

### 4. 运行验证

- **串口监视器**：115200 baud，查看系统日志
- **OLED显示**：实时显示温湿度、电压、电流、RFID等
- **蜂鸣器**：刷卡提示 / 温湿度超阈值报警

## 🔧 配置重置

长按 Boot 按钮 **3秒** → 蜂鸣两次 → 清除配置 → 重启进入配网模式

## 📡 数据传输

### TCP 连接

- **方式**：长连接，中途不断开
- **间隔**：每 3 秒发送一次
- **默认服务器**：192.168.1.100:8080

### JSON 数据格式

```json
{
  "number": 25824,
  "soc": 15,
  "temp": 20.7,
  "humidity": 77.1,
  "battery_id": "BAT2026001",
  "production_date": "2026-03-17",
  "cycle_count": 46,
  "ina_voltage": 2.17,
  "ina_current": 0.001,
  "ina_power": 3.125
}
```

**字段说明：**

| 字段 | 来源 | 说明 |
|------|------|------|
| number | 配置 | 设备ID |
| soc | 计算 | 电量(%) |
| temp/humidity | SHT3X | 温度(℃)/湿度(%) |
| battery_id/production_date/cycle_count | RFID卡片 | 电池信息 |
| ina_voltage/current/power | INA226 | 电压(V)/电流(A)/功率(W) |

## 📎 硬件接线表

### MFRC522 RFID模块（SPI）

| 引脚 | ESP32 | 说明 |
|------|-------|------|
| 3.3V | 3.3V | **禁止接5V** |
| GND | GND | 共地 |
| SCK | GPIO 18 | SPI时钟 |
| MISO | GPIO 19 | SPI数据 |
| MOSI | GPIO 23 | SPI数据 |
| SDA | GPIO 5 | 片选 |
| RST | GPIO 27 | 复位 |

### SHT3X 温湿度传感器（I2C1）

| 引脚 | ESP32 | 说明 |
|------|-------|------|
| VCC | 3.3V | 供电 |
| GND | GND | 共地 |
| SDA | GPIO 32 | 数据线 |
| SCL | GPIO 33 | 时钟线 |

### OLED 显示屏（I2C1，与SHT3X共用）

| 引脚 | ESP32 | 说明 |
|------|-------|------|
| VCC | 3.3V | 供电 |
| GND | GND | 共地 |
| SDA | GPIO 32 | 共用数据线 |
| SCL | GPIO 33 | 共用时钟线 |

### INA226 电流传感器（I2C2）

| 引脚 | ESP32 | 说明 |
|------|-------|------|
| VCC | 3.3V | 供电 |
| GND | GND | 共地 |
| SDA | GPIO 21 | 数据线 |
| SCL | GPIO 22 | 时钟线 |
| VBUS | 电池正极 | 电压测量 |
| IN+ | 电池正极 | 电流流入 |
| IN- | 负载正极 | 电流流出 |

### 其他组件

| 组件 | 引脚 | ESP32 | 说明 |
|------|------|-------|------|
| 蜂鸣器 | I/O | GPIO 26 | 低电平触发 |
| Boot按钮 | 按键 | GPIO 0 | 内置按钮 |

## 🖥️ 系统架构

```
ESP32 主控
├── SPI 总线  → MFRC522 RFID（电池身份识别）
├── I2C1 总线 → SHT3X（温湿度）+ OLED（显示）
├── I2C2 总线 → INA226（电流电压）
├── GPIO      → 蜂鸣器
└── WiFi      → TCP服务器（数据上传）
```

## 🎯 核心功能

| 功能 | 实现 | 说明 |
|------|------|------|
| RFID识别 | MFRC522，读块4/5/6 | <1秒响应 |
| 温湿度监测 | SHT3X | 每2秒采样 |
| 温湿度报警 | 蜂鸣器 | 阈值可配置 |
| 电流电压测量 | INA226 | 分流电阻可配置 |
| 数据上传 | TCP长连接 | 每3秒 |
| 配置存储 | EEPROM | 自动保存 |

## 🔧 故障排除

| 问题 | 检查项 |
|------|--------|
| 无法配网 | Boot按钮是否正常 |
| WiFi连接失败 | SSID/密码、信号强度 |
| TCP连接失败 | 服务器IP/端口、防火墙 |
| 传感器异常 | 接线、供电（3.3V） |
| OLED无显示 | I2C地址0x3C、GPIO32/33 |
| RFID不读卡 | SPI接线、卡片类型 |
| 电流测量异常 | 分流电阻值配置是否正确 |

## 🔒 安全注意事项

- MFRC522 只能用 3.3V
- 所有组件必须共地
- 分流电阻值必须与实际硬件匹配
- 配置页面密码建议修改（当前：12345678）
