# RFID+WiFi智能电池管理系统

## 📋 系统功能概述

### 1. RFID身份识别与数据管理
- 内置唯一RFID芯片标识电池身份，确保电池-设备-换电站的精准匹配
- 换电时快速校验，自动拦截错装、混装，保障换电安全与兼容性
- 支持读取卡片UID、卡片类型信息和SAK值
- 支持从卡片读取电池编号、生产日期和循环充电次数
- **每刷一次卡，自动增加循环充电次数**，实现电池使用次数的自动统计

### 2. WiFi数据上传
- 实时上传电池关键数据：SOC（剩余电量）、电压、电流、温度、湿度、RFID身份、电池编号、生产日期、循环充电次数等
- 采用长连接TCP通信，确保电池-换电站-云端数据稳定交互

### 3. 状态监测与预警
- 实时在线监测电池状态与环境数据
- 温湿度超阈值自动报警（温度>25°C或湿度≥90%），通过蜂鸣器提示
- 电池充电状态检测，通过LED直观指示充电状态

### 4. 本地可视化显示
- OLED显示屏（128x64）实时显示电池电压、电流、电量、温湿度、WiFi状态、RFID信息和充电状态
- 直观展示系统运行状态，方便现场查看
- 充电状态LED指示，充电时点亮，非充电时熄灭

### 5. 便捷配置与管理
- 自动配网功能，支持WiFi扫描和快速配置
- 长按Boot按钮重置配置，方便设备维护

### 6. 全生命周期追溯
- 建立电池唯一档案，通过设备ID和RFID实现身份识别
- 记录电池使用数据，便于质量追溯与维护决策

## 设备使用说明

### 🚀 首次使用步骤

1. **硬件连接**：按照最下方的附件中的接线表连接所有硬件组件

    **电池正极 → 120kΩ 电阻 → GPIO 34 **  

    **电池负极 → 24kΩ 电阻→ ESP32 GND**  

2. **上传代码**：将代码上传到 ESP32 开发板

3. **配网模式**：
   - 设备首次启动时，会自动进入配网模式
   - 打开手机或电脑的 WiFi 设置，连接名为 "Device-Config" 的热点
   - 热点密码：12345678

4. **配置设备**：
   - 连接热点后，打开浏览器，会自动跳转到配置页面
   - 填写 WiFi 名称、密码、服务器 IP 和端口
   - 点击 "保存配置" 按钮

5. **设备重启**：
   - 配置保存后，设备会自动重启
   - 重启后，设备会连接到配置的 WiFi 网络
   - 设备会每 10 秒发送一次数据到指定的 TCP 服务器

### 🔄 重新配置方法

如果需要重新配置设备，长按 Boot 按钮约 3 秒，设备会：
- 发出两次长蜂鸣声
- 清除现有配置
- 自动重启并进入配网模式

### 📡 数据传输

#### TCP 连接
- 采用长连接方式，中途不断开
- 服务器地址和端口可在配置页面设置

#### JSON 数据格式
```json
{
  "number": "25824",      // 设备ID（可在配置页面设置）
  "voltage": "3.05",      // 电池电压（V）
  "soc": "95",            // 电池电量（%）
  "temp": "25.5",         // 温度（℃）
  "humidity": "45.0",     // 湿度（%）
  "rfid": "A1B2C3D4",     // RFID 卡片ID
  "battery_id": "BAT2026001", // 电池编号
  "production_date": "2026-03-17", // 生产日期
  "cycle_count": "0",     // 循环充电次数
  "ina_voltage": "3.05",   // INA219 测量的电压（V）
  "ina_current": "0.500",  // INA219 测量的电流（A）
  "ina_power": "1.500"     // INA219 测量的功率（W）
}
```

### 🔧 故障排除

1. **无法进入配网模式**：
   - 检查 Boot 按钮是否正常
   - 尝试重新上传代码

2. **WiFi 连接失败**：
   - 检查 WiFi 名称和密码是否正确
   - 确保 WiFi 信号强度足够

3. **TCP 连接失败**：
   - 检查服务器 IP 和端口是否正确
   - 确保服务器正在运行并监听指定端口

4. **传感器数据异常**：
   - 检查传感器接线是否正确
   - 确保传感器供电正常

### 📝 注意事项

- 请勿将 MFRC522 模块直接连接到 5V 电源，否则会烧毁模块
- 确保所有组件的 GND 都正确连接
- 电池检测电路必须使用 120kΩ 和 24kΩ 的电阻分压
- 配置完成后，设备会自动保存配置到 EEPROM，重启后无需重新配置

### 📎 附件：硬件接线表

#### 1. MFRC522 刷卡模块（SPI 总线）

| MFRC522 引脚 | ESP32 引脚 | 说明 |
| --- | --- | --- |
| **3.3V** | **3.3V** | 供电（**切勿接 5V**，否则烧模块） |
| **GND** | **GND** | 共地 |
| **SCK** | **GPIO 18** | SPI 时钟 |
| **MISO** | **GPIO 19** | SPI 主入从出 |
| **MOSI** | **GPIO 23** | SPI 主出从入 |
| **SDA (SS)** | **GPIO 5** | 片选信号 |
| **RST** | **GPIO 27** | 复位引脚 |

#### 2. SHT30 / SHT31 温湿度传感器（独立 I2C_1 总线）

| SHT3X 引脚 | ESP32 引脚 | 说明 |
| --- | --- | --- |
| **VCC** (或 VIN) | **3.3V** | 供电 |
| **GND** | **GND** | 共地 |
| **SDA** | **GPIO 32** | 独立数据线 |
| **SCL** | **GPIO 33** | 独立时钟线 |

#### 3. 有源蜂鸣器 (Buzzer)

| 蜂鸣器 引脚 | ESP32 引脚 | 说明 |
| --- | --- | --- |
| **VCC** | **3.3V** 或 **5V** | 看你的蜂鸣器支持多少伏 |
| **GND** | **GND** | 共地 |
| **I/O** (信号) | **GPIO 26** | 触发引脚（低电平触发） |

#### 4. OLED 显示屏 (128x64, I2C 总线)

| OLED 引脚 | ESP32 引脚 | 说明 |
| --- | --- | --- |
| **VCC** | **3.3V** | 供电 |
| **GND** | **GND** | 共地 |
| **SDA** | **GPIO 32** | I2C 数据线（与 SHT3X 共用） |
| **SCL** | **GPIO 33** | I2C 时钟线（与 SHT3X 共用） |

#### 5. INA219 电流传感器（独立 I2C_2 总线）

| INA219 引脚 | ESP32 引脚 | 说明 |
| --- | --- | --- |
| **VCC** | **3.3V** | 供电 |
| **GND** | **GND** | 共地 |
| **SDA** | **GPIO 21** | I2C2 数据线（独立总线） |
| **SCL** | **GPIO 22** | I2C2 时钟线（独立总线） |
| **Vin+** | **电池正极** | 电池正极输入 |
| **Vin-** | **负载正极** | 负载正极输出 |

**注意：**
- GPIO32 和 33 是 SHT3X 温湿度传感器使用的 I2C1 总线引脚
- INA219 使用的是独立的 I2C2 总线，通过 GPIO21 和 22 连接

#### 6. 充电状态LED

| LED 引脚 | ESP32 引脚 | 说明 |
| --- | --- | --- |
| **VCC** | **3.3V** | 供电 |
| **GND** | **GND** | 共地 |
| **信号** | **GPIO 2** | 充电状态指示（充电时点亮） |

#### 7. 待测电池检测 (分压网络)

| 连接路径 | 说明 |
| --- | --- |
| **电池正极 → 120kΩ 电阻 → GPIO 34 ** | 分压网络，用于测量电池电压 |
| **电池负极 → 24kΩ 电阻→ ESP32 GND** | **非常重要：必须与 ESP32 共地！** |

### 📊 数据流程

1. **传感器数据采集**：
   - RFID 模块读取卡片信息
   - SHT3X 传感器读取温湿度
   - INA219 传感器读取电流电压
   - 分压网络读取电池电压

2. **数据处理**：
   - 计算电池电量（SOC）
   - 检测充电状态
   - 检测温湿度是否超过阈值

3. **数据显示**：
   - 在 OLED 显示屏上实时显示各项数据
   - 通过充电状态 LED 指示充电状态

4. **数据上传**：
   - 每 10 秒将数据打包为 JSON 格式
   - 通过 TCP 长连接发送到服务器

5. **用户交互**：
   - 刷卡时蜂鸣器提示
   - 温湿度超阈值时蜂鸣器警报
   - 长按 Boot 按钮重置配置

### 🔒 安全注意事项

- 所有组件必须正确接地，避免静电损坏
- MFRC522 模块只能使用 3.3V 供电，切勿接 5V
- 电池检测电路必须使用指定的电阻值，确保测量准确性
- 配置页面应设置强密码，防止未授权访问
- 服务器应设置访问控制，确保数据安全
